1)Для работы создать в директории с программой файл 
   Param.txt
В нем задать IP адресс, номер порта, колл-во каналов и имена для подключения. Пример:

ip:172.10.17.207 port:6666 channal:3 name:РАз,два,три;
ip:172.10.17.207 port:6667 channal:1 name:PlataTest;

2)Чтобы подключить базу данных имен создать файл 
  INI.txt 
в нем задать полный путь к базе данных. Пример:
C:\dbFol\db_.dat
