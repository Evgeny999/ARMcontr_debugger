1)Для работы создать в директории с программой файл 
   Param.txt
В нем задать IP адресс, номер порта, кол-во каналов и имена для подключения. 
Пример:
ip:172.10.17.207 port:6666 channal:3 name:РАз,два,три;
ip:172.10.17.207 port:6667 channal:1 name:PaltaTest;

2)Чтобы подключить базу данных имен:
В файле  Param.txt записать 
db:Путь к базе;
Пример:
db:C:\1111\db_.dat;
